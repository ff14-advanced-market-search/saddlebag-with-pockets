# saddlebag

## Development

Install dependencies w/ yarn:

```sh
yarn install
```

From your terminal:

```sh
yarn run dev
```

This starts your app in development mode, rebuilding assets on file changes.
Check your terminal for the address, the default is `http://127.0.0.1:8788`
